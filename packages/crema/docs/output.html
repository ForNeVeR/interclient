Output Parameters

Parameter Effect Notes
-d directory Specifies an alternative output directory for the obfuscated classes. Crema will create subdirectories for packages as required.
By default, Crema writes obfuscated classes to the same directory as the original classes.		 Use together with "-e .class"
-e .ext Specifies an alternative extension for the obfuscated classes. By default, Crema uses the .crema extension.  
-o Overwrite output files that already exist. Be careful not to overwrite original .class files!
