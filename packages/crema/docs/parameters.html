Command Line Parameters

Use command line parameters to tell Crema which classes to obfuscate, how to obfucate them, and where to put the result.

  • General parameters control general aspects of Crema's operation.
  • Input related parameters specify which .class files to process.
  • Output related parameters specify where to put the obfuscated classes.
  • Obfuscation parameters control the amount of obfuscation applied.
