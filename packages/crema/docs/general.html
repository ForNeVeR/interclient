General Parameters

Parameter Effect Notes
-v Turns on verbose mode: Crema reports which classes it is loading, obfuscating and saving.  
-map Outputs a table that maps original to obfuscated names. This table enables you to interpret stack tracebacks.
-@ fileName Directs Crema to read command line parameters from the specified file. See Parameter Files for more information.
