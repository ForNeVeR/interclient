Running Crema

You run Crema by telling the Java Virtual Machine to execute class Crema.

If you have: Start Crema this way:
JDK for Unix or Windows Type at a command prompt:
java Crema
optionally followed by command line parameters.
JDK for Macintosh Drop Crema.class on JavaRunner.
A dialog prompts you for command line parameters.
Microsoft Visual J++ Type at a command prompt:
jview Crema
optionally followed by command line parameters.
