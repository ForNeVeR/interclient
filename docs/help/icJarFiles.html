Distributed Jar Files

interclient-core.jar The JDBC driver with only the default English text resources. No utilities are included in the core jar.
interclient-res.jar Locale specific resource bundles.
interclient-utils.jar Currently consists of the communications diagnostics utility.
interclient.jar JDBC driver, all resources, and utilities.

The Windows install will automatically append interclient.jar to your user's classpath. You must modify your classpath manually on Unix.
