Customizing Text Resources for your Locale

Resources may already be tailored for certain locales in the interclient-res.jar (and interclient.jar) distribution as interbase/interclient/Resources_*.class and interbase/interclient/utils/Resources_*.class. Instructions for adding an existing resource, or for creating a new resource for your locale, may be found in the API reference for class interbase.interclient.Resources and class interbase.interclient.utils.Resources
