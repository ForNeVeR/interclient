End User Requirements for InterClient

In order to run Java applets using InterClient, a client must use a JRE 1.1 or 1.2 enabled browser.

The Java Runtime Environment (JRE) is installed with many popular Web browsers including the Netscape Navigator. The JRE is also available from Sun Microsystems.

The InterServer middleware server is included with the InterClient distribution and may be deployed on a Windows NT, Windows 95, Solaris, Linux, or HP-UX system. InterServer communicates directly with the InterBase server. All configurations require InterBase 5.0 or higher.

The Sun JRE 1.2 can be found at http://www.javasoft.com/products/jdk/1.2/jre
