System Configuration

On both Windows and Unix, the following entry must be available in the services file.

  • interserver 3060/tcp

The TCP/IP services file will have been modified as a part of the installation, if the installation defaults were accepted. If not, you may add these lines manually. The services file is located as follows in a default installation:
Windows NT \winnt\system32\drivers\etc\services
Windows 95 \windows\services
Unix /etc/services or an NIS services map.
In addition, an entry in the /etc/inetd.conf file is also required:
interserver stream tcp nowait root /usr/interclient/bin/interserver interserver

These configurations should be performed automatically during installation on Windows. For Unix installation instructions, see Install.txt

The CLASSPATH environment variable should be appended to include the interclient.jar from the root installation directory. This is done automatically by the Windows installation, but must be performed manually after installation on Unix.
