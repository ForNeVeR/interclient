JDBC Basics

Ok, let's leave architecture behind and talk about the jdbc api.

Simply put, ...

JDBC is an interface (API) for

  • Connecting to databases
  • Submitting commands
  • Processing results

A jdbc driver provides the code to support the jdbc interfaces

Basic JDBC interfaces are

  • java.sql.Connection
  • java.sql.Statement
  • java.sql.ResultSet

These interfaces are implemented by a driver.
