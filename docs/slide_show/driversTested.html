Drivers Tested

  • JDK 1.1 ODBC Bridge/InterBase 5
    Uses the InterBase net protocol thru ODBC.
  • InterClient 1.50/InterBase 5
    Uses a JDBC net protocol to ISC gateway (InterServer).
  • jConnect 4/SQLAnywhere 5
    Uses the Tabular Data Stream (TDS) net protocol.
  • DataGateway Broker 1.0/InterBase 5
    Uses a JDBC net protocol to BDE gateway.
    DG version which ships with JBuilder 2.
  • DataGateway Bridge 0.73/InterBase 5
    Uses the InterBase net protocol thru BDE.
    DG version which ships with JBuilder 2.
  • dbANYWHERE 1.1a/SQLAnywhere 5
    Uses a JDBC net protocol to ODBC gateway.
  • Driver X/Server X (not Oracle)
    Purported [by a newsgroup poster] to be the fastest JDBC driver.
